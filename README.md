# Platinum Custom Apparel Links Page

This version is closer to the glam neon mockup.

## Easy edits

Open `config.js` to change:
- Main links
- Social links
- Payment links
- Review link
- Button text
- Services
- Tagline/description/footer
- Number of floating hearts and sparkles

## Logo

Replace `assets/logo.png`.

## Contact card

Edit `contact.vcf`.
