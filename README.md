# Platinum Custom Apparel Exact Mockup Links Page

This version uses the approved mockup image as the page background so it looks exactly like the design.

## Edit links

Open `config.js` and replace the `#` placeholders.

Example:

```js
instagram: "#",
```

Change to:

```js
instagram: "https://instagram.com/yourusername",
```

## Edit the visual design

Replace:

`assets/mockup.png`

with a new design image using the same size/proportions.

## Save Contact

Edit:

`contact.vcf`

with your real phone and email.
