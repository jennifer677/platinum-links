# Platinum Custom Apparel Links Page

This version is the large glowing style with floating hearts and an easy-edit config file.

## Edit links later

Open `config.js`.

Replace the `#` placeholders with your real links.

Example:
instagram: "#"

Change to:
instagram: "https://instagram.com/yourname"

## Change logo

Replace:
assets/logo.png

## Save Contact button

Edit:
contact.vcf
