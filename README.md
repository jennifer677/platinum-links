# Platinum Custom Apparel Links Page

This version is easy to edit later.

## Main file to edit

Edit `config.js`.

That file controls:
- Main button links
- Social media links
- Payment links
- Google review link
- Tagline
- Description
- Services
- Floating hearts on/off
- Number of floating hearts

## Logo

Replace `assets/logo.png` if you want to change the logo.

## Contact card

Edit `contact.vcf` when you are ready to add your phone and email.
